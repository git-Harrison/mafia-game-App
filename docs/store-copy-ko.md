# 도트 마피아 — 스토어 등록 카피 초안 (한국어)

App Store / Google Play 제출용 한국어 메타데이터. 출시 전 디렉터 최종 검수 후 사용.

## 1. 앱 이름

**도트 마피아**

- iOS `CFBundleDisplayName`: 도트 마피아
- Android `android:label`: 도트 마피아

## 2. 한 줄 소개 (스토어 상단 헤드라인)

> 친구들과 함께 즐기는 픽셀 도트 마피아 게임.

대안:

- 광장에서 의심하고, 밤에는 움직인다.
- 4~8명, 한 판 10분, 픽셀 도트 마피아.

## 3. 짧은 설명 (Google Play short, 최대 80자)

> 친구 4~8명이 모여 한 판 10분 동안 즐기는 픽셀 도트 마피아 추리 게임.

## 4. 긴 설명 (App Store description / Google Play full)

```
어둠 속에서 누군가 움직인다.
당신은 누구를 의심하시겠어요?

도트 마피아는 친구 4~8명이 모여 즐기는 픽셀 도트 마피아 게임입니다.
낮에는 광장에 모여 토론하고, 밤에는 각자의 집에서 비밀스러운 행동이 일어납니다.
서로의 직업을 알아내고, 마피아를 찾아내거나, 시민을 속이세요.

▣ 게임 모드
- 친구 방 만들기 / 코드로 입장
- 4~8명, 한 판 10분 안팎
- 시민 · 경찰 · 의사 · 마피아 4가지 직업

▣ 게임 흐름
- 낮 토론에서 의심스러운 사람을 찾고
- 1차 투표로 후보를 정한 뒤
- 2차 찬반 표결로 처형 여부를 결정
- 밤에는 직업별 행동이 진행됩니다

▣ 특징
- 16비트 도트 그래픽 + 한국어 픽셀 폰트
- 가상 조이스틱으로 광장 자유 이동
- 페이즈마다 분위기 색이 바뀌는 밤
- 모바일에 맞춘 가벼운 한 판 진행

광장에서 친구를 시험하는 시간, 도트 마피아에서 시작해 보세요.
```

## 5. 주요 특징 (스토어 bullets / 5개)

1. 한 판 10분 안팎의 가벼운 픽셀 도트 마피아
2. 친구 코드로 방 만들기 + 입장 — 4~8명 동시 플레이
3. 시민 · 경찰 · 의사 · 마피아 4가지 직업
4. 낮 토론 · 1차 투표 · 2차 찬반 · 밤 3단계 직업 행동
5. 한국어 픽셀 폰트 + 도트 마을 분위기
